# CMPG311-Databases-Project-Group3
A collaborative CMPG311 database project showcasing ER diagrams, normalization, and SQL queries. Created for academic purposes and monitored by our assistant for evaluation

 CMPG 311 Database Project

Project Overview
This project is a Property Rental Listing Website with a working database.  
The system allows users to browse, list, and manage rental properties efficiently.

 Project Phases
- Phase 1→ System description and requirements documentation (`CMPG 311_Phase1 final (Updated).pdf`)
- Phase 2 → Entity-Relationship Diagram (ERD) design
- Phase 3 → SQL scripts for database creation and implementation
Objectives
- Define system requirements and database needs.
- Design a clear ERD to represent relationships between entities.
- Implement the database using SQL scripts.
